# Get-KB4AllowedDomain

## SYNOPSIS

Returns all allowed client domains.

## SYNTAX

```powershell
Get-KB4AllowedDomain [<CommonParameters>]
```

## DESCRIPTION

Returns all allowed client domains.

## EXAMPLES

### Example 1

```powershell
Get-KB4AllowedDomain
```

This command returns all current listed domains within the managed account.
